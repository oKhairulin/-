Объяснение кода:
get_exchange_rates: Функция отправляет GET-запрос к API и получает курсы валют. В данном примере используется API, который предоставляет курсы относительно доллара США (USD).
display_exchange_rates: Функция выводит на экран курсы валют в читаемом формате.
Основной цикл: В бесконечном цикле программа получает и отображает курсы валют, обновляя их каждые 10 секунд.
Примечание:
Замените URL API на тот, который Вы хотите использовать, и убедитесь, что у Вас есть доступ к API (возможно, потребуется API-ключ).
Убедитесь, что у Вас есть подключение к интернету для получения данных.
